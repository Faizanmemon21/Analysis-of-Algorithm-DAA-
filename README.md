# 🚚 Traveling Salesman Problem (TSP) - Heuristic Algorithm

A university project developed for the **Analysis of Algorithms** course. This project implements a **heuristic approach** to solve the Traveling Salesman Problem (TSP), providing an efficient approximate solution for finding the shortest route that visits each city exactly once and returns to the starting point.

---

## Features

- 🗺️ Traveling Salesman Problem (TSP)
- ⚡ Heuristic Algorithm Implementation
- 📈 Route Optimization
- 📊 Graph-Based Problem Solving
- 💻 Efficient Approximate Solution
